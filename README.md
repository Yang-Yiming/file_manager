# 🗂️ 文件快速访问器 v2.0 - 现代化版本

一个**高性能**的文件和文件夹快速访问工具，采用现代化UI设计，支持完整中文显示、智能标签管理、自动补全和高级设置功能。

## ✨ 新功能亮点 (v2.0)

### 🎨 全新现代化UI
- 🖼️ **卡片式设计**: 美观的阴影效果和圆角边框
- 🌈 **和谐配色**: 专业的色彩搭配，护眼舒适
- 📐 **精确对齐**: 所有按钮等高对齐，视觉统一
- 🔤 **中文优化**: 完美解决中文字符居中和缩进问题

### 🏷️ 智能标签系统
- ⚡ **实时补全**: 输入时自动显示历史标签建议
- #️⃣ **#标签格式**: 支持 `#工作 #重要` 格式，现代化标签体验
- 🔍 **快速搜索**: 点击标签立即筛选相关条目
- 🧠 **智能记忆**: 自动记录所有使用过的标签

### ⚙️ 高级设置界面
- 📁 **配置管理**: 可视化配置文件路径编辑
- 📊 **统计信息**: 实时显示条目数、标签数等统计
- 📤 **数据导出**: 一键导出备份数据
- 📥 **数据导入**: 从备份文件恢复数据
- 💾 **智能保存**: 自动保存所有配置更改

## 📖 使用指南

### 🚀 快速开始

```bash
# 克隆项目
git clone <repository-url>
cd file_manager

# 构建并运行
cargo run --release
```

### 🎯 核心功能

#### 1. 添加文件/文件夹
- **手动输入**: 在路径框中输入完整路径
- **浏览选择**: 点击 `📁 文件夹` 或 `📄 文件` 按钮
- **拖拽添加**: 直接将文件/文件夹拖到窗口中

#### 2. 智能标签管理
```
支持格式：
- 传统格式: 工作, 重要, 项目2024
- 现代格式: #工作 #重要 #项目2024
- 混合格式: #工作, 重要 #项目2024
```

- ⌨️ 输入时显示自动补全建议
- 🖱️ 点击建议快速选择
- 🔄 自动去重和排序

#### 3. 高效搜索
- 🔍 **实时搜索**: 输入即刻筛选结果
- 🎯 **多字段搜索**: 搜索路径、名称、描述、标签
- 🌏 **中文友好**: 完美支持中文搜索

#### 4. 设置管理
- ⚙️ 点击顶部 `⚙️ 设置` 按钮打开设置面板
- 📂 查看和修改配置文件路径
- 📈 查看实时统计信息
- 🔄 导入/导出数据备份

## 🏷️ 标签系统详解

### 标签格式支持
```bash
# 传统逗号分隔
工作, 重要, 项目2024, 文档

# 现代井号格式

# 混合格式
#工作, 重要 #项目2024, 文档
```

### 自动补全机制
- 🔤 **智能匹配**: 基于已有标签模糊匹配
- ⚡ **实时显示**: 输入过程中即时显示建议
- 🎯 **快速选择**: 点击建议自动补全
- 📝 **历史记录**: 记住所有使用过的标签

### 标签管理
- ➕ **自动收集**: 添加条目时自动更新标签库
- 🗑️ **智能清理**: 删除条目时清理无用标签
- 🔄 **实时同步**: 编辑标签时实时更新全局标签库

## 💾 配置文件格式

```json
{
  "entries": [
    {
      "path": "/Users/用户/Documents/重要项目",
      "name": "重要项目文档",
      "description": "包含所有重要的项目文档和资料",
      "tags": ["工作", "重要", "项目2024"],
      "is_directory": true,
      "created_at": "2024-05-28T21:08:23.123456789+08:00"
    }
  ]
}
```
